Project TempCheck

The system utilizes Rasberry Pi and DS18B20 temperature sensor for monitoring the temperature data as well as control the status of the temperature sensor remotely. The required sensor data can be transferred to an external storage device like PD and can also be sent to the cloud so as to preserve the on-chip memory.
